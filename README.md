Assignment 02 – Resource Allocation Under Uncertainty

Ryan Recio
Optimization-based marketing budget allocation using prescriptive analytics and sensitivity analysis.

Notes

Context chosen: Marketing (ad spend allocation across five channels).

Synthetic data generation: A language model (LLM) was used to generate a 5-option synthetic dataset including conversions per $1k, cost per $1k, minimum and maximum spend constraints, and a total budget constraint of $100,000.

The optimization model was built using PuLP in Python and includes sensitivity analysis, a what-if scenario, validation checks, and a stakeholder-driven constraint update.
